<h2>HP Otto: Construction Otto</h2>

STL files and build instructions can be found on [printables.com](https://www.printables.com/model/1484511-hp-otto-construction-otto)

rc.py is a modified version of rc.py enabling control of Construction Otto via the new HP Otto phone app <i>(currently (Mar 2026) in beta test and only available to purchasers of the HP Otto Starter https://www.facebook.com/groups/ottodiy/posts/3923882564421879/)</i>  
A timer is used for continuous movement of the servo until control button is released.  
Additions and modifications to the base rc.py are all commented so that future releases of rc.py can be similarly modified.  

<img width="512" height="229" alt="construction otto phone control" src="https://github.com/user-attachments/assets/8640ee29-934d-4771-97b1-03a7b65168b6" />

 The joystick control controls movement in any given direction, the D-pad control controls the lifting arm and the gripper.

<h3>Instructions</h3>  
Just drop rc.py in Otto's root directory.
