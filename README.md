<h1>HP Otto Varients</h1>

This repository is for code that is specific to certain HP Otto varients.
It is assumed that the robots will already have the basic HP Otto libraries.

<img width="262" height="208" alt="Gripper Otto" src="https://github.com/user-attachments/assets/faa580ed-2bad-4655-ad05-c89f35bffae6" />

Construction Otto has a lifting mechanism terminating in a universal tool mount. 
A variety of different tools can be used with this mount
